# ClawTris — Game Design Document

## Обзор

**Жанр:** Puzzle / Arcade  
**Платформа:** PC (Windows, Linux, macOS)  
**Движок:** Godot 4.x  
**Статус:** В разработке

## Концепция

Классический Тетрис с современным неоновым визуальным стилем, плавными эффектами и удобным управлением для ПК.

## Визуальный стиль

### Цветовая палитра
| Роль | Цвет | Hex |
|------|------|-----|
| Фон | Почти чёрный | #0a0a0f |
| Primary | Cyan | #00ffff |
| Secondary | Magenta | #ff00ff |
| Accent | Purple | #9d00ff |
| Success | Neon Green | #00ff88 |

### Характеристики
- Glow эффекты на UI и фигурах
- Частицы при очистке линий
- Анимированные переходы
- Современный геометрический шрифт

## Управление (PC)

| Действие | Клавиши |
|----------|---------|
| Влево | ← / A |
| Вправо | → / D |
| Soft drop | ↓ / S |
| Hard drop | Space |
| Поворот | ↑ / W / X |
| Пауза | ESC |

## Игровые экраны

1. **Main Menu** — Play, Settings, Quit
2. **Game** — игровое поле, счёт, следующая фигура
3. **Pause** — Resume, Restart, Main Menu
4. **Game Over** — финальный счёт, Restart, Main Menu
5. **Settings** — громкость, управление (будущее)

## Механики

- Классические правила Тетриса
- 7 типов тетромино (I, O, T, S, Z, J, L)
- Ghost piece (тень)
- Система уровней (ускорение)
- Очки: 100 × линии × уровень
